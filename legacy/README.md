# Legacy

Referensmaterial från den första versionen av Samify CRM (vanilla JS + Supabase, enfils-SPA).
Sparat för att underlätta migrationen till React.

- `index.html` — huvudapplikationen (4043 rader)
- `samify-status.html` — publik delbar statussida för kunder

Dessa filer **används inte** av den nya React-appen. När migrationen är helt klar och verifierad kan `legacy/` tas bort.
